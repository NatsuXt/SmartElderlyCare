## 前端集成建议
前端在调用API时必须设置正确的Content-Type头：
```javascript
fetch('http://47.96.238.102:3003/api/RoomManagement/rooms', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json; charset=utf-8'
    },
    body: JSON.stringify({
        roomType: "豪华套房",
        bedType: "双人床",
        status: "可用"
    })
});
```

**重要提醒**: charset=utf-8 是关键，缺少这个参数会导致中文字符乱码！
