## 快速部署步骤

### 1. 环境检查
```powershell
# 检查网络连接
.\network_check.bat

# 检查端口占用
netstat -ano | findstr ":3003"
```

### 2. 权限设置 (首次部署必须)
```powershell
# 以管理员身份运行
.\setup_permissions.bat
```

### 3. 启动服务器
```powershell
# 方式一: 安全启动（推荐）
.\start_server_safe.bat

# 方式二: PowerShell脚本启动
.\start_server_fixed.ps1

# 方式三: 直接启动
.\RoomDeviceManagement.exe --urls="http://*:3003"
```

### 4. 验证部署
```powershell
# 检查API文档
.\find_swagger.ps1

# 或直接访问
http://47.96.238.102:3003/swagger
```
